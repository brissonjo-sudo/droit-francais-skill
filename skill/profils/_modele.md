# Profil — [intitulé]

> **Mode d'emploi.** Un profil configure les **défauts** du skill pour un
> praticien donné : contexte territorial, domaines prioritaires, 3ᵉ regard
> d'auto-critique (étape 7). Pour l'activer, copier ce fichier (ou l'un des
> profils fournis) en **`profil.md`** à la racine du skill installé :
>
> ```
> cp profils/police-gendarmerie.md profil.md
> ```
>
> **Rien ici n'est une certitude** : toute valeur est un défaut qu'une
> requête précise surcharge, et que l'étape 0 bis remet en question dès
> qu'elle devient décisionnelle. Sans `profil.md`, le skill tourne en
> **profil neutre** (aucune hypothèse métier/territoriale présumée).
> Remplacer chaque `[à compléter]`.
