# Profil — Juriste d'entreprise

> Défauts pour un juriste d'entreprise / direction juridique. Copier en
> `profil.md` pour activer. Remplacer les `[à compléter]`. Valeurs = défauts.

## 1. Identité d'usage
- **Qui** : juriste d'entreprise, direction juridique.
- **Livrables typiques** : note d'analyse, avis de conformité, revue
  contractuelle, cartographie de risque, support à la décision.
- **Niveau d'exigence par défaut** : note de fond.

## 2. Contexte territorial par défaut
- **Siège / implantations** : [à compléter].
- **Droit applicable** : droit français ; signaler les points relevant du
  droit de l'UE (à vérifier séparément) ou de contrats internationaux.
- **Particularités** : [secteur régulé ? — à compléter].

## 3. Domaines prioritaires (pilote la veille — §10)
| Domaine / code | Priorité |
|----------------|----------|
| Droit des sociétés (C. com.) | haute |
| Droit social (C. travail, conventions collectives KALI) | haute |
| Conformité / RGPD / réglementation sectorielle | haute |
| Droit des contrats (C. civ.) | moyenne |
| Concurrence, commande publique (selon activité) | variable |

## 4. Rôle (c) d'auto-critique (étape 7)
- **3ᵉ regard métier** : **le régulateur / commissaire aux comptes / auditeur**.
- **Question type** : « Cette position tiendrait-elle face à un contrôle
  (AMF, CNIL, URSSAF, DGCCRF…) ou à un audit ? Où est le risque de
  requalification ou de sanction ? »

## 5. Cas particuliers métier
- **Conventions collectives (KALI)** : identifier le numéro **IDCC** ;
  vérifier la version en vigueur de la convention **et** de l'accord
  d'entreprise applicable.
- **Articulation loi / contrat** : distinguer dispositions impératives et
  supplétives.
- **Droit de l'UE** : vérifier la transposition en droit français
  séparément (règlements d'application directe vs directives).
- **Sanctions administratives à caractère punitif** : régime proche du
  pénal (P6, jurisprudence *Engel*).
